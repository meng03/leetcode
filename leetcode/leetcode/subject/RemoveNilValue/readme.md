
### 描述
递归遍历集合，移除集合中的空值

### 示例

````
处理前
[
"sub1":["a","b",nil,"d"],
"sub2":nil,
"sub3":[
"sub1":["a","b",nil,"d"],
"sub2":nil,
"sub3":2
],
"sub4": [["hh": 34],["gg":4],["kk":nil]]
]
处理后
[
"sub4": [["hh": 34], ["gg": 4]], 
"sub1": ["a", "b", "d"], 
"sub3": ["sub1": ["a", "b", "d"], 
"sub3": 2]
]
````
